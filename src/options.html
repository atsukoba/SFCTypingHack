SFC Typing Hack

SFC Typing Hackはあなたのタイピングを支援するどころか全てを代わりに実行します。

View GitHub Report errors or bugs Contact Author

 

圧倒的なスピード

最大で約毎分15000打鍵のスピードで、且つ完璧な精度であなたのタイピング試験練習問題をこなします。

人間らしさ

高速自動ミススペル機能により決められた割合のタイプを失敗することで、"人間らしい"自動タイピングを実現します。

オープンソース

ソースコード含めすべて公開しています。あなたによる改造を待っています。

Parameters

Name Dsc Value
Speed タイピングの速度(type/s)を指定します。 [5, 50, 100, 200, 500]
Error Rate 自動ミスタイプの割合を指定します [0, 0.5%, 1%, 5%]
